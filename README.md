## Descripción
Mini proyecto con el objetivo de reforzar conocimientos aprendidos en la curso de Ciencia de Datos de Alura. En este challenge se toma una empresa que ha tenido una pérdida de clientes, donde se deben de realizar análisis para determinas posibles causas y qué se puede considerar hacer para mantener y atraer a nuevos clientes.
